# AI Image Generator

A simple and interactive AI Image Generator built using **Python**, **Streamlit**, and the **Pollinations AI API**. Users can generate AI-powered images from text prompts, customize image dimensions, apply enhanced prompt quality, and download the generated images.

## ✨ Features

* 🎨 Generate AI images from text prompts
* 🖌️ Choose from multiple art styles:

  * Photorealistic
  * Anime
  * Vintage Victorian
  * Sketch
  * 3D Render
* 📏 Adjustable image width and height
* ✨ **Magic Enhance** mode for higher-quality image generation
* 🎲 **Surprise Me** button with random creative prompts
* 📥 Download generated images as PNG files

## 🛠️ Technologies Used

* Python

* Pollinations AI Image API

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/AI-Image-Generator.git
```

### 2. Navigate to the project folder

```bash
cd AI-Image-Generator
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
streamlit run app.py
```

---

## 📸 How to Use

1. Enter a text prompt describing the image.
2. Select an art style.
3. Adjust the image width and height.
4. (Optional) Enable **Magic Enhance** for more detailed results.
5. Click **Generate Image**.
6. Or click **🎲 Surprise Me** to generate a random creative image.
7. Download the generated image as a PNG file.
